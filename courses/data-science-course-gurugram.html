Master Data Science & Machine Learning Course Syllabus

Our comprehensive curriculum is split into logical modules to guide you from fundamentals to advanced skills.

  • Main Topic 1: Python Syntax & Data Structures — Variables, Control Flow, List Comprehensions, Dictionaries, Lambda Functions
  • Main Topic 2: Object-Oriented Programming (OOP) — Classes, Objects, Inheritance, Encapsulation, Custom Modules
  • Sub-topic: File Handling & Formats — Parsing CSV, JSON, Excel, Exception Handling & Virtual Environments
  • Practical Capstone: Building a modular automated data ingestion pipeline in Python
  • Main Topic 1: Descriptive Statistics & Probability — Mean, Median, Variance, Standard Deviation, Normal & Binomial Distributions
  • Main Topic 2: Inferential Statistics & Hypothesis Testing — Z-test, T-test, ANOVA, Chi-Square, p-values, Confidence Intervals
  • Sub-topic: Business A/B Testing — Designing experiments, sample size estimation, and conversion rate analysis
  • Practical Capstone: Statistical hypothesis testing report on e-commerce user behavior
  • Main Topic 1: Multi-Dimensional Data Arrays (NumPy) — Matrix operations, indexing, vectorization & broadcast rules
  • Main Topic 2: Advanced Pandas Data Manipulations — DataFrames, Missing Value Imputation, GroupBy, Merging, Pivot Tables
  • Sub-topic: Database Querying with SQL — Joins, Subqueries, CTEs, Window Functions (`RANK`, `DENSE_RANK`)
  • Practical Capstone: Cleaning and wrangling large scale financial transaction datasets
  • Main Topic 1: Exploratory Visual Analytics — Line plots, Histograms, Box plots, Scatter charts, Heatmaps (Seaborn)
  • Main Topic 2: Interactive BI Dashboards with Power BI — Power Query ETL, Star Schema Modeling, DAX Measures
  • Practical Capstone: Creating an interactive executive dashboard published to Power BI Cloud
  • Main Topic 1: Supervised ML Algorithms — Linear & Logistic Regression, Decision Trees, Random Forests, XGBoost, SVM
  • Main Topic 2: Unsupervised Learning & Dimensionality Reduction — K-Means Clustering, Principal Component Analysis (PCA)
  • Sub-topic: Feature Engineering & Tuning — One-Hot Encoding, StandardScaler, GridSearchCV, Cross-Validation
  • Sub-topic: MLOps Deployment & REST APIs — Serving models via FastAPI/Flask REST endpoints & Streamlit Web Apps
  • Practical Capstone: End-to-End Predictive Machine Learning Application Deployed on Cloud
  • Real-world cases: E-commerce customer churn and product recommendations
  • Final Portfolio: Completing a deployment-ready capstone project
